# Heiman Home για το HomeAssistant

---
![logo@2x.png](custom_components/heiman_home/brand/logo%402x.png)
## Επισκόπηση

Αυτή η ενσωμάτωση για το Home Assistant επιτρέπει τη χρήση συσκευών έξυπνου σπιτιού Heiman σε ένα σύστημα έξυπνου σπιτιού. Συνδέεται με το Heiman Cloud API και χρησιμοποιεί το πρωτόκολλο MQTT για ενημερώσεις κατάστασης συσκευών σε πραγματικό χρόνο.

Αυτή η ενσωμάτωση επιτρέπει στους χρήστες να ενσωματώσουν εύκολα τις συσκευές Heiman τους στο Home Assistant και να τις χρησιμοποιήσουν για διάφορες αυτοματοποιήσεις και παρακολούθηση, συμπεριλαμβανομένης της διαχείρισης firmware και του ελέγχου θυγατρικών συσκευών.

## Εγκατάσταση


### Μέθοδος 1: HACS (Συνιστάται)

#### Πρώτη Εγκατάσταση
1. Ανοίξτε **HACS** → **Ενσωματώσεις**
2. Κάντε κλικ στο **+ ΕΞΕΡΕΥΝΗΣΗ & ΛΗΨΗ ΑΠΟΘΕΤΗΡΙΩΝ**
3. Αναζητήστε `Heiman Home` ή κάντε κλικ στις τρεις τελείες (⋮) → **Προσαρμοσμένα αποθετήρια**
4. Προσθήκη αποθετηρίου: `https://github.com/hass-user/heiman-home` με κατηγορία `Integration`
5. Κάντε κλικ στο **ΛΗΨΗ ΑΥΤΟΥ ΤΟΥ ΑΠΟΘΕΤΗΡΙΟΥ**

#### Ενημέρωση Component
1. Ανοίξτε **HACS** → **Ενσωματώσεις**
   ![Download (1)](https://github.com/Elwinmage/ha-xsense-component/assets/15807572/3220c686-f53f-4766-9523-e3272a6ff104)
2. Βρείτε το **Heiman Home**
   ![img.png](image/img.png)
3. Κάντε κλικ στο ****ΕΝΗΜΕΡΩΣΗ** ή **Επαναλήψη λήψης****
   ![img_1.png](image/img_1.png)
### Μέθοδος 2: Χειροκίνητη Εγκατάσταση μέσω Samba/SFTP

1. Κατεβάστε την latest έκδοση από τα [GitHub Releases](https://github.com/hass-user/heiman-home/releases)
2. Αποσυμπιέστε τον φάκελο `heiman_home`
3. Αντιγράψτε τον φάκελο `heiman_home` στον κατάλογο `custom_components` του Home Assistant σας
   ```
   /config/custom_components/heiman_home/
   ```
![img_2.png](image/img_2.png)
![img_3.png](image/img_3.png)
### Μέθοδος 3: One-key Shell μέσω SSH/Terminal & SSH Add-on

```shell
wget -O - https://raw.githubusercontent.com/hass-user/heiman-home/main/install.sh | bash -
```

### Μετά την Εγκατάσταση

1. Κάντε επανεκκίνηση του Home Assistant
2. Μεταβείτε στις **Ρυθμίσεις** → **Συσκευές και Υπηρεσίες**
3. Κάντε κλικ στο **+ Προσθήκη Ενσωμάτωσης**
4. Αναζητήστε `Heiman Home`

<a name="configuration"></a>
## Διαμόρφωση

### Προσθήκη Ενσωμάτωσης μέσω Web UI

1. Ανοίξτε το Home Assistant web UI
2. Μεταβείτε στις **Ρυθμίσεις** → **Συσκευές και Υπηρεσίες**
3. Κάντε κλικ στο **Προσθήκη Ενσωμάτωσης**
4. Αναζητήστε `Heiman Home` και επιλέξτε το
5. Εξουσιοδοτήστε με τον λογαριασμό Heiman σας (OAuth2)
6. Επιλέξτε το σπίτι που θέλετε να ενσωματώσετε
7. Ολοκληρώστε τη ρύθμιση

### Πιστοποίηση

Η ενσωμάτωση χρησιμοποιεί OAuth2 για ασφαλή πιστοποίηση:
- Κάντε κλικ στο **Εξουσιοδότηση** για να συνδεθείτε στον λογαριασμό Heiman σας
- Παραχωρήστε τις απαραίτητες άδειες
- Επιλέξτε το σπίτι που θέλετε να ενσωματώσετε

### Πολλαπλά Σπίτια

Μπορείτε να προσθέσετε πολλαπλά σπίτια Heiman:
1. Κάθε σπίτι δημιουργεί μια ξεχωριστή καταχώρηση διαμόρφωσης
2. Κάθε σπίτι έχει ανεξάρτητη διαχείριση συσκευών
3. Διαμορφώστε κάθε σπίτι ξεχωριστά στις **Συσκευές και Υπηρεσίες**

### Φιλτράρισμα Συσκευών

Μπορείτε να φιλτράρετε ποιες συσκευές θα ενσωματωθούν:
1. Ανοίξτε τις ρυθμίσεις ενσωμάτωσης
2. Μεταβείτε στις **Επιλογές**
3. Ενεργοποιήστε/απενεργοποιήστε συγκεκριμένες συσκευές
4. Αποθηκεύστε τις αλλαγές

<a name="firmware-management"></a>
## Διαχείριση Firmware

### Έλεγχος για Ενημερώσεις
- Οι ενημερώσεις firmware ελέγχονται αυτόματα κατά τον συγχρονισμό συσκευών
- Οι οντότητες ενημέρωσης εμφανίζουν διαθέσιμες εκδόσεις firmware
- Συγκρίνετε την εγκατεστημένη έκδοση με την τελευταία διαθέσιμη έκδοση

### Εγκατάσταση Ενημερώσεων Firmware
1. Μεταβείτε στη συσκευή στο Home Assistant
2. Ανοίξτε την οντότητα **Firmware**
3. Κάντε κλικ στο κουμπί **Εγκατάσταση**
4. Παρακολουθήστε την πρόοδο ενημέρωσης σε πραγματικό χρόνο
5. Η συσκευή θα επανεκκινηθεί μετά την ολοκλήρωση της ενημέρωσης

### Λειτουργίες Ενημέρωσης
- ✅ Αυτόματη ανίχνευση έκδοσης
- ✅ Παρακολούθηση προόδου (0-100%)
- ✅ Παρακολούθηση κατάστασης ενημέρωσης
- ✅ Σύγκριση εκδόσεων
- ✅ Έλεγχος μαζικών ενημερώσεων

<a name="mqtt-integration"></a>
## Ενσωμάτωση MQTT

### Ενημερώσεις σε Πραγματικό Χρόνο
Η ενσωμάτωση χρησιμοποιεί MQTT για άμεσες ενημερώσεις κατάστασης συσκευών:
- Καμία καθυστέρηση polling
- Άμεσες αλλαγές κατάστασης
- Χαμηλότερη χρήση API
- Καλύτερη απόδοση

### Διαμόρφωση MQTT
- Διαμορφώνεται αυτόματα κατά τη ρύθμιση
- Χρησιμοποιεί τον MQTT broker της Heiman
- Ασφαλής σύνδεση με TLS/SSL
- Δεν απαιτείται χειροκίνητη διαμόρφωση

### Υποστηριζόμενες Λειτουργίες MQTT
- Ενημερώσεις ιδιοτήτων συσκευής
- Κατάσταση online/offline
- Γεγονότα συναγερμού
- Αναγνώσεις αισθητήρων
- Διαχείριση θυγατρικών συσκευών (εγγραφή, απεγγραφή, ανακάλυψη)

<a name="child-device-management"></a>
## Διαχείριση Θυγατρικών Συσκευών

Η ενσωμάτωση παρέχει ολοκληρωμένη διαχείριση θυγατρικών συσκευών μέσω της βιβλιοθήκης `heiman-connect`.
Όλη η λογική διαχείρισης συσκευών υλοποιείται στο SDK και η ενσωμάτωση Home Assistant παρέχει ένα απλό API για πρόσβαση σε αυτές τις λειτουργίες.

### Χρήση του Child Device Manager

```python
from heimanconnect import ChildDeviceManager

# Λήψη child device manager από τον API client
device_manager = await api_client.async_get_child_device_manager(
    user_id="your_user_id",
    devices=devices_dict,
    user_display_name="Your Name"
)

# Προσθήκη θυγατρικής συσκευής (συνιστώμενη μέθοδος)
result = await device_manager.add_and_sync_device(
    gateway_product_id="1733421468953686016",
    gateway_device_id="1760910156165971969",
    child_device_id="1768080341172985856",
    child_product_id="1734821218500292608",
    child_device_name="Door Sensor"
)

# Αφαίρεση θυγατρικής συσκευής (συνιστώμενη μέθοδος)
result = await device_manager.remove_and_sync_device(
    gateway_product_id="1733421468953686016",
    gateway_device_id="1760910156165971969",
    child_device_id="1768080341172985856",
    product_id="1734821218500292608",
    device_name="01000053"
)
```

### Διαθέσιμες Μέθοδοι

Για λεπτομερή τεκμηρίωση όλων των διαθέσιμων μεθόδων και των παραμέτρων τους, δείτε την [τεκμηρίωση της βιβλιοθήκης heiman-connect](https://pypi.org/project/heiman-connect/).

<a name="advanced-configuration"></a>
## Προηγμένη Διαμόρφωση

### Διαμόρφωση Logger

Ενεργοποίηση debug logging για αντιμετώπιση προβλημάτων:

```yaml
# configuration.yaml
logger:
  default: warning
  logs:
    custom_components.heiman_home: debug
    heimanconnect: debug
```

### Προσαρμογή Οντοτήτων

```yaml
# configuration.yaml
homeassistant:
  customize: !include customize.yaml

# customize.yaml
sensor.heiman_temperature:
  friendly_name: "Living Room Temperature"
  device_class: temperature
  unit_of_measurement: "°C"

binary_sensor.heiman_door:
  friendly_name: "Front Door"
  device_class: door
```

### Εξαίρεση Ιδιοτήτων

```yaml
# configuration.yaml
heiman_home:
  exclude_attributes:
    - raw_data
    - firmware_info
    - configuration
```

<a name="debugging"></a>
## Αποσφαλμάτωση

### Λήψη Ιδιοτήτων Κατάστασης Οντότητας

1. Ανοίξτε **Εργαλεία Προγραμματιστών** → **Καταστάσεις**
2. Αναζητήστε την οντότητά σας (π.χ., `sensor.heiman_temperature`)
3. Προβολή όλων των ιδιοτήτων και της τρέχουσας κατάστασης

### Λήψη Debug Logs

1. Ενεργοποίηση debug logging (δείτε [Διαμόρφωση Logger](#advanced-configuration))
2. Ανοίξτε **Ρυθμίσεις** → **Σύστημα** → **Αρχεία Καταγραφής**
3. Αναζητήστε `heiman_home` ή `heimanconnect`

### Συχνά Προβλήματα

#### Η Συσκευή Δεν Εμφανίζεται
- Ελέγξτε τις ρυθμίσεις φιλτραρίσματος συσκευών
- Βεβαιωθείτε ότι η συσκευή είναι online στην εφαρμογή Heiman
- Κάντε επανεκκίνηση της ενσωμάτωσης

#### Η Ενημέρωση Firmware Δεν Λειτουργεί
- Βεβαιωθείτε ότι η συσκευή είναι online
- Ελέγξτε τη συμβατότητα της συσκευής
- Βεβαιωθείτε ότι υπάρχει διαθέσιμη ενημέρωση firmware στην εφαρμογή Heiman

#### Προβλήματα Σύνδεσης
- Ελέγξτε τη σύνδεση internet
- Επαληθεύστε τα διαπιστευτήρια λογαριασμού Heiman
- Ελέγξτε τα αρχεία καταγραφής του Home Assistant για σφάλματα

#### Το MQTT Δεν Συνδέεται
- Βεβαιωθείτε ότι το δίκτυο επιτρέπει εξερχόμενες συνδέσεις στο `spmqtt.heiman.cn:1884`
- Ελέγξτε τις ρυθμίσεις firewall
- Κάντε επανεκκίνηση του Home Assistant

<a name="troubleshooting"></a>
## Αντιμετώπιση Προβλημάτων

### Αποτυχία Πιστοποίησης
1. Εξουσιοδοτήστε ξανά την ενσωμάτωση
2. Επαληθεύστε τα διαπιστευτήρια λογαριασμού Heiman
3. Ελέγξτε αν ο λογαριασμός έχει πρόσβαση στο επιλεγμένο σπίτι

### Οι Συσκευές Δεν Ενημερώνονται
1. Ελέγξτε την κατάσταση σύνδεσης MQTT στα αρχεία καταγραφής
2. Βεβαιωθείτε ότι η συσκευή είναι online
3. Δοκιμάστε χειροκίνητη ανανέωση μέσω υπηρεσίας

### Υψηλή Χρήση Βάσης Δεδομένων
- Εξαιρέστε unnecessary attributes (δείτε [Εξαίρεση Ιδιοτήτων](#advanced-configuration))
- Απενεργοποιήστε unused entities
- Ελέγξτε για entities με πάρα πολλές αλλαγές κατάστασης

### Προβλήματα Απόδοσης
- Μειώστε το διάστημα ενημέρωσης εάν είναι δυνατόν
- Φιλτράρετε τις unused devices
- Απενεργοποιήστε το MQTT εάν δεν χρειάζεται (δεν συνιστάται)

## Ανάπτυξη

### Δομή Έργου
```
heiman_home/
├── __init__.py          # Αρχικοποίηση ενσωμάτωσης
├── api.py               # Wrapper API client
├── config_flow.py       # Ροή διαμόρφωσης
├── const.py             # Σταθερές και διαμόρφωση
├── coordinator.py       # Συντονιστής ενημέρωσης δεδομένων
├── sensor.py            # Πλατφόρμα αισθητήρα
├── binary_sensor.py     # Πλατφόρμα δυαδικού αισθητήρα
├── switch.py            # Πλατφόρμα διακόπτη
├── button.py            # Πλατφόρμα κουμπιού
├── select.py            # Πλατφόρμα επιλογέα
├── number.py            # Πλατφόρμα αριθμού
├── update.py            # Πλατφόρμα ενημέρωσης (firmware)
├── manifest.json        # Μεταδεδομένα ενσωμάτωσης
└── strings.json         # Μεταφράσεις
```

**Σημείωση**: Όλη η λογική διαχείρισης συσκευών βρίσκεται στη βιβλιοθήκη `heiman-connect`, όχι σε αυτή την ενσωμάτωση.
