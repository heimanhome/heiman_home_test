![logo@2x.png](custom_components/heiman_home/brand/logo%402x.png)
## अवलोकन

यह होम असिस्टेंट इंटीग्रेशन स्मार्ट होम सिस्टम में हेइमान स्मार्ट होम डिवाइसों का उपयोग करने की अनुमति देता है। यह हेइमान क्लाउड API से कनेक्ट होता है और रीयल-टाइम डिवाइस स्टेटस अपडेट के लिए MQTT प्रोटोकॉल का उपयोग करता है।

यह इंटीग्रेशन उपयोगकर्ताओं को अपने हेइमान डिवाइसों को होम असिस्टेंट में आसानी से एकीकृत करने और फर्मवेयर प्रबंधन और चाइल्ड डिवाइस नियंत्रण सहित विभिन्न स्वचालन और निगरानी के लिए उनका उपयोग करने की अनुमति देता है।

- ☁️ **क्लाउड इंटीग्रेशन**: OAuth2 प्रमाणीकरण के साथ हेइमान खाते के माध्यम से कनेक्ट करें
- 📡 **MQTT रीयल-टाइम अपडेट**: MQTT पुश के माध्यम से त्वरित डिवाइस स्टेटस अपडेट

- 👨‍👩‍👧‍👦 **चाइल्ड डिवाइस प्रबंधन**: MQTT के माध्यम से गेटवे सब-डिवाइस जोड़ें, हटाएं और प्रबंधित करें
- 🏠 **बहु-घर समर्थन**: कई हेइमान घरों को स्वतंत्र रूप से प्रबंधित करें

- ⚙️ **वेब UI कॉन्फ़िगरेशन**: YAML कॉन्फ़िगरेशन के बिना आसान सेटअप

<a name="installation"></a>
## इंस्टॉलेशन


### विधि 1: HACS (अनुशंसित)

#### पहली बार इंस्टॉलेशन
1. खोलें **HACS** → **इंटीग्रेशन्स**
2. क्लिक करें **+ रिपॉजिटरीज़ एक्सप्लोर एंड डाउनलोड**
3. `Heiman Home` खोजें या तीन बिंदुओं (⋮) पर क्लिक करें → **कस्टम रिपॉजिटरीज़**
4. रिपॉजिटरी जोड़ें: `https://github.com/hass-user/heiman-home` श्रेणी `Integration` के साथ
5. क्लिक करें **डाउनलोड दिस रिपॉजिटरी**

#### घटक अपडेट करें
1. खोलें **HACS** → **इंटीग्रेशन्स**
   ![Download (1)](https://github.com/Elwinmage/ha-xsense-component/assets/15807572/3220c686-f53f-4766-9523-e3272a6ff104)
2. ढूंढें **Heiman Home**
   ![img.png](image/img.png)
3. क्लिक करें ****अपडेट** या **रीडाउनलोड****
   ![img_1.png](image/img_1.png)

3. `heiman_home` फ़ोल्डर को अपने होम असिस्टेंट `custom_components` निर्देशिका में कॉपी करें


3. क्लिक करें **+ ऐड इंटीग्रेशन**
4. `Heiman Home` खोजें

<a name="configuration"></a>
## कॉन्फ़िगरेशन

### वेब UI के माध्यम से इंटीग्रेशन जोड़ें

1. होम असिस्टेंट वेब इंटरफेस खोलें
2. **सेटिंग्स** → **डिवाइसेज़ एंड सर्विसेज़** पर नेविगेट करें
3. **ऐड इंटीग्रेशन** पर क्लिक करें
4. `Heiman Home` खोजें और उसे चुनें
5. अपने हेइमान खाते (OAuth2) के साथ अधिकृत करें
6. उस घर का चयन करें जिसे आप एकीकृत करना चाहते हैं
7. सेटअप पूरा करें

### प्रमाणीकरण

इंटीग्रेशन सुरक्षित प्रमाणीकरण के लिए OAuth2 का उपयोग करता है:
- अपने हेइमान खाते में लॉगिन करने के लिए **ऑथोराइज़** पर क्लिक करें
- आवश्यक अनुमतियां दें
- उस घर का चयन करें जिसे आप एकीकृत करना चाहते हैं

### कई घर

आप कई हेइमान घर जोड़ सकते हैं:
1. प्रत्येक घर एक अलग कॉन्फ़िगरेशन प्रविष्टि बनाता है
2. प्रत्येक घर में स्वतंत्र डिवाइस प्रबंधन होता है
3. **डिवाइसेज़ एंड सर्विसेज़** में प्रत्येक घर को अलग से कॉन्फ़िगर करें

### डिवाइस फ़िल्टरिंग

आप फ़िल्टर कर सकते हैं कि कौन से डिवाइस एकीकृत किए जाएं:
1. इंटीग्रेशन सेटिंग्स खोलें
2. **ऑप्शन्स** पर नेविगेट करें
3. विशिष्ट डिवाइसों को सक्षम/अक्षम करें
4. परिवर्तनों को सहेजें

## फर्मवेयर प्रबंधन

### अपडेट खोजें
- डिवाइस सिंक्रनाइज़ेशन के दौरान फर्मवेयर अपडेट स्वचालित रूप से जांचे जाते हैं

### फर्मवेयर अपडेट इंस्टॉल करें
1. होम असिस्टेंट में डिवाइस पर नेविगेट करें

4. रीयल-टाइम में अपडेट प्रोग्रेस को मॉनिटर करें
5. अपडेट पूरा होने के बाद डिवाइस रीस्टार्ट होगा

### अपडेट सुविधाएं

- ✅ वर्जन तुलना
- ✅ बैच अपडेट चेक

<a name="mqtt-integration"></a>
## MQTT इंटीग्रेशन

### रीयल-टाइम अपडेट
इंटीग्रेशन त्वरित डिवाइस स्टेटस अपडेट के लिए MQTT का उपयोग करता है:
- कोई पोलिंग देरी नहीं
- तात्कालिक स्टेटस परिवर्तन
- कम API उपयोग
- बेहतर प्रदर्शन

### MQTT कॉन्फ़िगरेशन
- सेटअप के दौरान स्वचालित रूप से कॉन्फ़िगर किया गया
- हेइमान MQTT ब्रोकर का उपयोग करता है
- TLS/SSL के साथ सुरक्षित कनेक्शन
- कोई मैनुअल कॉन्फ़िगरेशन आवश्यक नहीं

### समर्थित MQTT सुविधाएं
- डिवाइस प्रॉपर्टी अपडेट
- ऑनलाइन/ऑफलाइन स्टेटस
- अलार्म इवेंट्स
- सेंसर रीडिंग
- चाइल्ड डिवाइस प्रबंधन (पंजीकरण, अपंजीकरण, खोज)

<a name="child-device-management"></a>
## चाइल्ड डिवाइस प्रबंधन

इंटीग्रेशन `heiman-connect` लाइब्रेरी के माध्यम से व्यापक चाइल्ड डिवाइस प्रबंधन प्रदान करता है।
सभी डिवाइस प्रबंधन लॉजिक SDK में लागू किया गया है, और होम असिस्टेंट इंटीग्रेशन इन सुविधाओं तक पहुंचने के लिए एक सरल API प्रदान करता है।

### चाइल्ड डिवाइस मैनेजर का उपयोग

```python
from heimanconnect import ChildDeviceManager

# API क्लाइंट से चाइल्ड डिवाइस मैनेजर प्राप्त करें
device_manager = await api_client.async_get_child_device_manager(
    user_id="your_user_id",
    devices=devices_dict,
    user_display_name="Your Name"
)

# चाइल्ड डिवाइस जोड़ें (अनुशंसित विधि)
result = await device_manager.add_and_sync_device(
    gateway_product_id="1733421468953686016",
    gateway_device_id="1760910156165971969",
    child_device_id="1768080341172985856",
    child_product_id="1734821218500292608",
    child_device_name="Door Sensor"
)

# चाइल्ड डिवाइस हटाएं (अनुशंसित विधि)
result = await device_manager.remove_and_sync_device(
    gateway_product_id="1733421468953686016",
    gateway_device_id="1760910156165971969",
    child_device_id="1768080341172985856",
    product_id="1734821218500292608",
    device_name="01000053"
)
```

### उपलब्ध विधियां

सभी उपलब्ध विधियों और उनके पैरामीटर के विस्तृत दस्तावेज़ीकरण के लिए, [heiman-connect लाइब्रेरी दस्तावेज़ीकरण](https://pypi.org/project/heiman-connect/) देखें।

<a name="advanced-configuration"></a>
## उन्नत कॉन्फ़िगरेशन

### लॉगर कॉन्फ़िगरेशन

ट्रबलशूटिंग के लिए डीबग लॉगिंग सक्षम करें:

```yaml
# configuration.yaml
logger:
  default: warning
  logs:
    custom_components.heiman_home: debug
    heimanconnect: debug
```

### इकाइयों को अनुकूलित करें

```yaml
# configuration.yaml
homeassistant:
  customize: !include customize.yaml

# customize.yaml
sensor.heiman_temperature:
  friendly_name: "Living Room Temperature"
  device_class: temperature
  unit_of_measurement: "°C"

binary_sensor.heiman_door:
  friendly_name: "Front Door"
  device_class: door
```

### विशेषताएं बाहर करें

```yaml
# configuration.yaml
heiman_home:
  exclude_attributes:
    - raw_data
    - firmware_info
    - configuration
```

<a name="services"></a>
## सेवाएं

### `heiman_home.refresh_device`

किसी डिवाइस की स्थिति को मैनुअली रिफ्रेश करें।

```yaml
service: heiman_home.refresh_device
data:
  device_id: "1972646416676724736"
```

### `heiman_home.refresh_all_devices`

इंटीग्रेशन में सभी डिवाइसों को रिफ्रेश करें।

```yaml
service: heiman_home.refresh_all_devices
```

## डीबगिंग

### इकाई स्टेटस विशेषताएं प्राप्त करें

1. खोलें **डेवलपर टूल्स** → **स्टेट्स**
2. अपनी इकाई खोजें (उदाहरण के लिए, `sensor.heiman_temperature`)
3. सभी विशेषताओं और वर्तमान स्थिति को देखें

### डीबग लॉग प्राप्त करें

2. खोलें **सेटिंग्स** → **सिस्टम** → **लॉग्स**
3. `heiman_home` या `heimanconnect` खोजें

### सामान्य समस्याएं

#### डिवाइस दिखाई नहीं दे रहा है
- डिवाइस फ़िल्टर सेटिंग्स की जांच करें
- सुनिश्चित करें कि डिवाइस हेइमान ऐप में ऑनलाइन है
- इंटीग्रेशन को रीस्टार्ट करें

#### फर्मवेयर अपडेट काम नहीं कर रहा है
- सुनिश्चित करें कि डिवाइस ऑनलाइन है
- डिवाइस संगतता की जांच करें
- सुनिश्चित करें कि हेइमान ऐप में फर्मवेयर अपडेट उपलब्ध है

#### कनेक्शन समस्याएं
- इंटरनेट कनेक्शन की जांच करें
- हेइमान खाता क्रेडेंशियल्स की जांच करें
- त्रुटियों के लिए होम असिस्टेंट लॉग्स की जांच करें

#### MQTT कनेक्ट नहीं हो रहा है
- सुनिश्चित करें कि नेटवर्क `spmqtt.heiman.cn:1884` पर आउटगोइंग कनेक्शन की अनुमति देता है
- फ़ायरवॉल सेटिंग्स की जांच करें
- होम असिस्टेंट को रीस्टार्ट करें

<a name="troubleshooting"></a>
## समस्या निवारण

### प्रमाणीकरण विफल
1. इंटीग्रेशन को फिर से अधिकृत करें
2. हेइमान खाता क्रेडेंशियल्स की जांच करें
3. सुनिश्चित करें कि खाते में चयनित घर तक पहुंच है

### डिवाइस अपडेट नहीं हो रहे हैं
1. लॉग्स में MQTT कनेक्शन स्टेटस की जांच करें
2. सुनिश्चित करें कि डिवाइस ऑनलाइन है
3. सेवा के माध्यम से मैनुअल रिफ्रेश का प्रयास करें

### उच्च डेटाबेस उपयोग
- अनावश्यक विशेषताओं को बाहर करें (देखें [विशेषताएं बाहर करें](#advanced-configuration))
- अप्रयुक्त इकाइयों को अक्षम करें
- बहुत अधिक स्टेटस परिवर्तनों वाली इकाइयों की जांच करें

### प्रदर्शन समस्याएं
- यदि संभव हो तो अपडेट अंतराल कम करें
- अप्रयुक्त डिवाइसों को फ़िल्टर करें
- यदि आवश्यक न हो तो MQTT अक्षम करें (अनुशंसित नहीं)

<a name="development"></a>
## विकास

### प्रोजेक्ट संरचना
```
heiman_home/
├── __init__.py          # इंटीग्रेशन इनिशियलाइज़ेशन
├── api.py               # API क्लाइंट रैपर
├── config_flow.py       # कॉन्फ़िगरेशन फ्लो
├── const.py             # स्थिरांक और कॉन्फ़िगरेशन
├── coordinator.py       # डेटा अपडेट कोऑर्डिनेटर
├── sensor.py            # सेंसर प्लेटफॉर्म
├── binary_sensor.py     # बाइनरी सेंसर प्लेटफॉर्म
├── switch.py            # स्विच प्लेटफॉर्म
├── button.py            # बटन प्लेटफॉर्म
├── select.py            # चयनकर्ता प्लेटफॉर्म
├── number.py            # संख्या प्लेटफॉर्म
├── update.py            # अपडेट प्लेटफॉर्म (फर्मवेयर)
├── manifest.json        # इंटीग्रेशन मेटाडेटा
└── strings.json         # अनुवाद
```

**नोट**: सभी डिवाइस प्रबंधन लॉजिक `heiman-connect` लाइब्रेरी में है, इस इंटीग्रेशन में नहीं।

### निर्भरताएं
- `heiman-connect`: हेइमान API के लिए Python लाइब्रेरी
- `packaging`: फर्मवेयर अपडेट के लिए वर्जन तुलना
