## نظرة عامة

تسمح هذه الإضافة لـ Home Assistant باستخدام أجهزة المنزل الذكي من Heiman ضمن نظام المنزل الذكي. تتصل بواجهة برمجة تطبيقات Heiman السحابية وتستخدم بروتوكول MQTT لتحديثات حالة الجهاز في الوقت الفعلي.

تمكن هذه الإضافة المستخدمين من دمج أجهزة Heiman الخاصة بهم بسهولة في Home Assistant واستخدامها لمختلف الأتمتة والمراقبة، بما في ذلك إدارة البرامج الثابتة والتحكم في الأجهزة الفرعية.

<a name="installation"></a>
## التثبيت


### الطريقة 1: HACS (موصى به)

#### التثبيت الأول
1. افتح **HACS** → **الإضافات**
2. انقر على **+ استكشاف وتنزيل المستودعات**
3. ابحث عن `Heiman Home` أو انقر على النقاط الثلاث (⋮) → **المستودعات المخصصة**
4. أضف المستودع: `https://github.com/hass-user/heiman-home` مع الفئة `Integration`
5. انقر على **تنزيل هذا المستودع**

#### تحديث المكون
1. افتح **HACS** → **الإضافات**
   ![Download (1)](https://github.com/Elwinmage/ha-xsense-component/assets/15807572/3220c686-f53f-4766-9523-e3272a6ff104)
2. ابحث عن **Heiman Home**
   ![img.png](image/img.png)
3. انقر على ****تحديث** أو **إعادة التنزيل****
   ![img_1.png](image/img_1.png)
### الطريقة 2: التثبيت اليدوي عبر Samba/SFTP

1. قم بتنزيل أحدث إصدار من [إصدارات GitHub](https://github.com/hass-user/heiman-home/releases)
2. استخرج مجلد `heiman_home`
3. انسخ مجلد `heiman_home` إلى دليل `custom_components` في Home Assistant الخاص بك
   ```
   /config/custom_components/heiman_home/
   ```
![img_2.png](image/img_2.png)
![img_3.png](image/img_3.png)
### الطريقة 3: Shell بنقرة واحدة عبر SSH/Terminal و SSH Add-on

```shell
wget -O - https://raw.githubusercontent.com/hass-user/heiman-home/main/install.sh | bash -
```

### بعد التثبيت

1. أعد تشغيل Home Assistant
2. انتقل إلى **الإعدادات** → **الأجهزة والخدمات**
3. انقر على **+ إضافة إضافة**
4. ابحث عن `Heiman Home`

<a name="configuration"></a>
## التكوين

### إضافة الإضافة عبر واجهة الويب

1. افتح واجهة ويب Home Assistant
2. انتقل إلى **الإعدادات** → **الأجهزة والخدمات**
3. انقر على **إضافة إضافة**
4. ابحث عن `Heiman Home` وحدده
5. قم بالتفويض باستخدام حساب Heiman الخاص بك (OAuth2)
6. حدد المنزل الذي تريد دمجه
7. أكمل الإعداد

### المصادقة

تستخدم الإضافة OAuth2 للمصادقة الآمنة:
- انقر على **تفويض** لتسجيل الدخول إلى حساب Heiman الخاص بك
- امنح الأذونات اللازمة
- حدد المنزل الذي تريد دمجه

### منازل متعددة

يمكنك إضافة عدة منازل من Heiman:
1. ينشئ كل منزل إدخال تكوين منفصل
2. يحتوي كل منزل على إدارة أجهزة مستقلة
3. قم بتكوين كل منزل بشكل منفصل في **الأجهزة والخدمات**

### تصفية الأجهزة

يمكنك تصفية الأجهزة التي تريد دمجها:
1. افتح إعدادات الإضافة
2. انتقل إلى **الخيارات**
3. تمكين/تعطيل أجهزة محددة
4. احفظ التغييرات

4. راقب تقدم التحديث في الوقت الفعلي
5. سيعاد تشغيل الجهاز بعد اكتمال التحديث

### ميزات التحديث
- ✅ الكشف التلقائي عن الإصدار
- ✅ مراقبة التقدم (0-100%)
- ✅ تتبع حالة التحديث
- ✅ مقارنة الإصدارات
- ✅ التحقق من التحديثات المجمعة

<a name="mqtt-integration"></a>
## تكامل MQTT

### تحديثات الوقت الفعلي
تستخدم الإضافة MQTT لتحديثات حالة الجهاز الفورية:
- لا يوجد تأخير في الاستطلاع
- تغييرات الحالة الفورية
- استخدام أقل لواجهة برمجة التطبيقات
- أداء أفضل

### تكوين MQTT
- يتم تكوينه تلقائيًا أثناء الإعداد
- يستخدم وسيط MQTT الخاص بـ Heiman
- اتصال آمن مع TLS/SSL
- لا يتطلب تكوينًا يدويًا

### ميزات MQTT المدعومة
- تحديثات خصائص الجهاز
- حالة الاتصال/عدم الاتصال
- أحداث الإنذار
- قراءات المستشعر
- إدارة الأجهزة الفرعية (التسجيل، إلغاء التسجيل، الاكتشاف)

<a name="child-device-management"></a>
## إدارة الأجهزة الفرعية

توفر الإضافة إدارة شاملة للأجهزة الفرعية من خلال مكتبة `heiman-connect`.
يتم تنفيذ جميع منطق إدارة الجهاز في SDK، وتوفر إضافة Home Assistant واجهة برمجة تطبيقات بسيطة للوصول إلى هذه الميزات.

### استخدام مدير الأجهزة الفرعية

```python
from heimanconnect import ChildDeviceManager

# الحصول على مدير الأجهزة الفرعية من عميل API
device_manager = await api_client.async_get_child_device_manager(
    user_id="your_user_id",
    devices=devices_dict,
    user_display_name="Your Name"
)

# إضافة جهاز فرعي (الطريقة الموصى بها)
result = await device_manager.add_and_sync_device(
    gateway_product_id="1733421468953686016",
    gateway_device_id="1760910156165971969",
    child_device_id="1768080341172985856",
    child_product_id="1734821218500292608",
    child_device_name="Door Sensor"
)

# إزالة جهاز فرعي (الطريقة الموصى بها)
result = await device_manager.remove_and_sync_device(
    gateway_product_id="1733421468953686016",
    gateway_device_id="1760910156165971969",
    child_device_id="1768080341172985856",
    product_id="1734821218500292608",
    device_name="01000053"
)
```

### الطرق المتاحة

للحصول على وثائق مفصلة حول جميع الطرق المتاحة ومعلماتها، راجع [وثائق مكتبة heiman-connect](https://pypi.org/project/heiman-connect/).

<a name="advanced-configuration"></a>
## التكوين المتقدم

### تكوين السجل

تمكين تسجيل التصحيح لاستكشاف الأخطاء وإصلاحها:

```yaml
# configuration.yaml
logger:
  default: warning
  logs:
    custom_components.heiman_home: debug
    heimanconnect: debug
```

### تخصيص الكيانات

```yaml
# configuration.yaml
homeassistant:
  customize: !include customize.yaml

# customize.yaml
sensor.heiman_temperature:
  friendly_name: "Living Room Temperature"
  device_class: temperature
  unit_of_measurement: "°C"

binary_sensor.heiman_door:
  friendly_name: "Front Door"
  device_class: door
```

### استبعاد السمات

```yaml
# configuration.yaml
heiman_home:
  exclude_attributes:
    - raw_data
    - firmware_info
    - configuration
```

<a name="debugging"></a>
## التصحيح

### الحصول على سمات حالة الكيان

1. افتح **أدوات المطور** → **الحالات**
2. ابحث عن كيانك (على سبيل المثال، `sensor.heiman_temperature`)
3. عرض جميع السمات والحالة الحالية

### الحصول على سجلات التصحيح

1. تمكين تسجيل التصحيح (راجع [تكوين السجل](#advanced-configuration))
2. افتح **الإعدادات** → **النظام** → **السجلات**
3. ابحث عن `heiman_home` أو `heimanconnect`

### المشاكل الشائعة

#### الجهاز لا يظهر
- تحقق من إعدادات تصفية الجهاز
- تأكد من أن الجهاز متصل بالإنترنت في تطبيق Heiman
- أعد تشغيل الإضافة

#### تحديث البرنامج الثابت لا يعمل
- تأكد من أن الجهاز متصل بالإنترنت
- تحقق من توافق الجهاز
- تأكد من توفر تحديث البرنامج الثابت في تطبيق Heiman

#### مشاكل الاتصال
- تحقق من اتصال الإنترنت
- تحقق من بيانات اعتماد حساب Heiman
- تحقق من سجلات Home Assistant بحثًا عن أخطاء

#### MQTT لا يتصل
- تأكد من أن الشبكة تسمح بالاتصالات الصادرة إلى `spmqtt.heiman.cn:1884`
- تحقق من إعدادات جدار الحماية
- أعد تشغيل Home Assistant

<a name="troubleshooting"></a>
## استكشاف الأخطاء وإصلاحها

### فشلت المصادقة
1. أعد تفويض الإضافة
2. تحقق من بيانات اعتماد حساب Heiman
3. تحقق مما إذا كان الحساب لديه حق الوصول إلى المنزل المحدد

### الأجهزة لا يتم تحديثها
1. تحقق من حالة اتصال MQTT في السجلات
2. تأكد من أن الجهاز متصل بالإنترنت
3. حاول التحديث اليدوي عبر الخدمة

### استخدام قاعدة البيانات العالي
- استبعد السمات غير الضرورية (راجع [استبعاد السمات](#advanced-configuration))
- عطل الكيانات غير المستخدمة
- تحقق من الكيانات ذات التغييرات الكثيرة جدًا في الحالة

### مشاكل الأداء
- قلل فترة التحديث إذا أمكن
- قم بتصفية الأجهزة غير المستخدمة
- عطل MQTT إذا لم يكن مطلوبًا (غير موصى به)
